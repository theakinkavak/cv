The commands/shortcuts to create multiple cursors, multiple selections and multiple lines in Atom:

  1. CTRL + SHIFT + D: To duplicate a line
  2. CTRL + Click: To create multiple cursors
  3. CTRL + ALT + (Up arrow key) or (Down arrow key): To create multiple cursors
  4. CTRL + D: Multiple selection
  5. ALT + F3: Multiple selection (Faster than CTRL + D)
